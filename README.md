# FunSocial-Zone

🌐 FunSocial Zone 🎉

FunSocial Zone is a modern, responsive, and interactive social media website designed to deliver a fun and engaging user experience 😄✨ This project focuses on front-end development, creative UI design, and smooth user interactions using core web technologies.

🚀 Live Website:
https://shahrishabh1513-jsk.github.io/FunSocial-Zone/

✨ Features:
📱 Fully responsive layout for desktop and mobile  
🌙 Dark-themed modern UI  
📰 Social media feed with posts and stories  
🔔 Interactive buttons and UI elements  
🎨 Clean, attractive, and user-friendly design  

🛠️ Technologies Used:
🔹 HTML – Structure and layout  
🔹 CSS – Styling, animations, and responsiveness  
🔹 JavaScript – Interactivity and dynamic behavior  

🎯 Project Purpose:
This project was built for learning, practice, and portfolio showcase purposes. It demonstrates front-end development skills, UI/UX understanding, and the ability to create visually appealing social media-style websites.

📂 How to Use:
1️⃣ Clone or download the repository  
2️⃣ Open index.html in any modern browser  
3️⃣ Explore the features of FunSocial Zone
